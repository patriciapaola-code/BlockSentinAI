Start scripts

Para facilitar o início da aplicação há scripts prontos na raiz do projeto:

- Unix/macOS:
```
./start.sh
```

- Windows (PowerShell / CMD):
```
start.bat
```

Os scripts ativam o `venv` (se existir), instalam dependências e iniciam o Streamlit.

Observação: se preferir, você também pode rodar diretamente:

```
streamlit run main.py
```
